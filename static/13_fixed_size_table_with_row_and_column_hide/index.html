An HTML table that;

  • allows column names and cell values to be edited in place
  • allows rows and columns to be hidden
    • hide columns with Alt + ↑ ↓
    • hide rows with Alt + → ←
    • maybe hidden isn't the right term because you can still see hidden rows and columns
    • the idea is that you might want to exclude some data from the JSON export but be able to quickly get the data back - so we don't want to delete the data from the table and make people re-enter it
  • supports export of data to JSON
    • hidden rows and columns are not included in this export
