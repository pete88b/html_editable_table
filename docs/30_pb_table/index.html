HTML5 editable table

An HTML table that;

  • can be re-built from JSON
  • allows column names and cell values to be edited in place
  • allows new columns and rows to be added
  • support navigation within the table via Ctrl + Arrow Key
  • supports export of data to JSON
  • allows columns and rows to be excluded from export, without being deleted from the table
